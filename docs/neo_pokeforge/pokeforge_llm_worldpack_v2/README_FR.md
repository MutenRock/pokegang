# Pokeforge LLM World Pack V2 (FR)

Pack bilingue de départ pour créer des **PNJ Pokémon pilotés par LLM** dans Pokeforge / Rocket HQ.

## Inclus

- **1025 espèces Pokémon uniques** (`data/pokedex/pokemon_1025_en.json`)
- **147 fiches de personnages nommés**
- **240 fiches de dresseurs procéduraux**
- **Schéma de mémoire PNJ + exemple**
- **Templates de prompt** en FR et EN
- **Scripts Python utilitaires**

## Important

C'est un **world pack de départ prêt pour la prod**, pas une encyclopédie canon exhaustive de tous les PNJ nommés de la franchise.

Tu obtiens :
- base espèces complète
- set solide de personnages nommés
- population procédurale scalable
- fondation mémoire / relations
