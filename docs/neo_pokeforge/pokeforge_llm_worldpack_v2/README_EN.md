# Pokeforge LLM World Pack V2 (EN)

A bilingual starter pack for building **Pokémon-inspired LLM NPC systems** inside Pokeforge / Rocket HQ.

## Included

- **1025 unique Pokémon species** (`data/pokedex/pokemon_1025_en.json`)
- **147 lore-ready named character sheets**
- **240 procedural trainer sheets**
- **NPC memory schema + example**
- **Prompt templates** in EN and FR
- **Utility Python scripts**

## Important

This is a **production-ready starter world pack**, not an exhaustive canon dump of every named trainer in franchise history.

You get:
- complete species base
- strong named-character starter set
- scalable procedural population
- memory system foundations
