# Vision

Pokeforge aims to create a persistent simulation playground where characters, creatures, and societies evolve over time.

The long-term goal is not to produce static quest chains, but to create a world where:

- actors remember meaningful events,
- relationships change,
- local conflicts emerge,
- language output feels contextual,
- progression can be observed or played directly.

LLMs should support expression, context framing, and micro-level reasoning. The underlying world model should remain inspectable and testable.
