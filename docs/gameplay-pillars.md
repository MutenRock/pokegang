# Gameplay pillars

1. **Persistence** — the world should keep state and history.
2. **Emergence** — interesting situations should arise from systems interacting.
3. **Readability** — the player must understand why something happened.
4. **Agency** — player input should matter at multiple scales.
5. **Modularity** — simulation, dialogue, and battle layers must stay separable.
6. **Bilingual UX** — English and French should both be first-class citizens.
